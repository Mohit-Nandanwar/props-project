# Richest Personalities Card Grid

## 📌 Project Overview
This project is a **React application** that displays information about the 12 richest personalities in the world.  
Each person is shown in a **card layout** with:
- Their **image**
- Their **name**
- A short **description**

The cards are styled with **animated gradient borders** and hover effects for a modern UI.

---

## 🛠️ Tech Stack
- **React.js** – Component-based UI
- **CSS** – Custom styling (`App.css`, `main.css`)
- **JavaScript (ES6+)** – Logic and rendering
